# Simple filter JQuery

#### Use exemple
```
(function ($) {
    var options = {
        'filterButton': 'sf-filter-button',
        'filterOptions': 'sf-option',
        'searchInput': 'sf-input',
        'searchButton': 'sf-search-button',
        'baseUrl': '/',
    };
    $('.filter-main-container').simpleFilter(options);
})(jQuery);
```
